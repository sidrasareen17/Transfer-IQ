# ⚽ TransferIQ – AI Player Intelligence Dashboard

## 📌 Project Overview
TransferIQ is an AI-powered football analytics dashboard that analyzes player performance and predicts market value using the Random Forest Regression model.  
It also includes sentiment analysis to understand public opinion about players.

---

## 🚀 Features
- 📊 Market Value Trend Graph
- 🤖 Prediction using Random Forest Regression
- 📈 Player Performance Analysis (Goals, Assists)
- 📉 Sentiment Analysis (VADER)
- 🎯 Interactive Dashboard with real-time graphs

---

## 🛠️ Technologies Used
- Python (Flask)
- Pandas, NumPy
- Scikit-learn (Random Forest)
- HTML, CSS, JavaScript
- Chart.js
